# Logistics Reporting Automation – PT PELNI Internship Project

This project was developed during my internship at PT PELNI (Persero) Surabaya Branch.

The system automates operational reporting for passenger and freight transport services by transforming raw passenger transaction data into structured monthly reports.

## Features
- Automated data processing using Python
- Logistics data grouping by voyage and cargo type
- Automated report generation using Google Apps Script

## Technologies
Python
Google Apps Script
Pdf
Excel
